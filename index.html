WeatherMajor - IEMBot Web Monitor and WeatherStar4000+

IEMBot

The WeatherMajor service provides real-time weather monitoring through IEMBot. It allows users to track weather messages organized by the NWS Office.

This service includes various tools such as the VTEC Product Browser, SPC Watch Overviews, River Summary App, and Local Storm Report Mapping.

WeatherStar4000+

WeatherStar4000+ (WS+) is a program by Michael Battaglia! The program is based on WeatherStar4000 (WS) by taiganet.com.

The web-based version of WS is still great, with mostly the same options as WS.
